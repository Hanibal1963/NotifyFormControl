# NotifyForm Control

Ein Control zum Anzeigen von Benachrichtigungsfenstern.

## Eigenschaften

- **Design** - Gibt das Design des Benachrichtigungsfensters an.
- **Message** - Der Text, der im Benachrichtigungsfenster angezeigt wird.
- **ShowTime** - Die Zeit, die das Benachrichtigungsfenster angezeigt wird, bevor es automatisch geschlossen wird.
- **Style** - Der Stil des Benachrichtigungsfensters, z.B. Information, Warnung, Fehler.
- **Title** - Der Titel des Benachrichtigungsfensters.
